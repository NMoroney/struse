
## ᔑႵ𐍂ߎᎦ६ : ६𐋄𐀁ጨꛠ⠧६Ꭶ

The struse processing begins with lower case [ASCII](https://en.wikipedia.org/wiki/ASCII) characters between a and z. For a single input character an [HTML entity](https://en.wikipedia.org/wiki/List_of_XML_and_HTML_character_entity_references) substitution is performed. 

For example for one of the dictionaries, the letter 'm' is replaced with ```&#x1328;``` or the HTML entity for the [Ethiopic syllable cha](https://en.wikipedia.org/wiki/Geʽez_script).

This HTML entity is then added to a web page or HTML file and rendered by a browser. For ```&#x1328;``` or the Ethiopic syllable cha the result is &#x1328;.

For input text, the above substitution is repeated for each character of input.

---

As an additional example, the following is a line from [Jean Joubert's](https://fr.wikipedia.org/wiki/Jean_Joubert) poem 'Wind Script':

> Il n'y a plus alors que des éclats de langue,

The resulting struse HTML entity substitutions :

```&#x1109;&#x2827;&ensp; &#x0E81;'&#x10284;&ensp; &#x10300;&ensp; &#xAA6D;&#x10539;&#x053F;&#x0D17;&ensp; &#x10300;&#x2827;&#x0DA7;&#x053B;&#x0D17;&ensp; &#x1007;&#x053F;&#x0B3D;&ensp; &#x1095;&#x0B3D;&#x0D17;&ensp; é&#x122D;&#x2827;&#x10001;&#x1D366;&#x1511;&ensp; &#x1D507;&#x0B3D;&ensp; &#x1D4DB;&#x1D70;&#x0E17;&#x0D6D;&#x00B5;&#x096C;,```

Finally, as rendered with your browser :
